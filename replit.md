# InsulaAI Landing Page

## Overview
This is a static landing page for InsulaAI - a UAV-based insulator inspection system powered by YOLOv8 deep learning. The page showcases the technology, features, and benefits of automated insulator fault detection using drone imagery and AI.

## Project Structure
- `index.html` - Main landing page with embedded CSS and JavaScript
- `server.py` - Simple Python HTTP server to serve the static content
- Port 5000 is used for the web server

## Technology Stack
- Static HTML/CSS/JavaScript
- Bootstrap 5 for responsive design
- GSAP for animations
- Python HTTP server for serving content

## Recent Changes
- 2025-11-04: Initial GitHub import and Replit environment setup
- Configured Python server on port 5000 with no-cache headers
- Set up workflow for automatic server startup

## Features
The landing page includes:
- Interactive custom cursor with particle effects
- Animated hero section with SVG drone illustration
- Feature cards highlighting AI capabilities
- Step-by-step "How It Works" section
- Live demo visualization of fault detection
- Technology stack showcase
- Responsive design for all devices

## Running the Project
The server automatically starts via the configured workflow. It serves the static HTML page on port 5000 and is accessible through the Replit webview.
